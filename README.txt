Custom console formatting are being used as running this in an IDE's output window won't format properly. 

To run the program, just click runSpaceBNB.bat to automatically compile and execute the program

(Optional)
For running manually:
  1. Open a terminal (preferably cmd) in the project's root directory
  2. To compile, enter command: javac -encoding UTF-8 -d out src\main\java\org\Group5\*.java
  3. To execute, enter command: java -Dfile.encoding=UTF-8 -cp out org.Group5.Main
